# 4.如何更新以上数据

> 对应Binance API:  https://developers.binance.com/docs/zh-CN/derivatives/usds-margined-futures/market-data/rest-api/Exchange-Information

```bash
python trade/multi/exchange_info.py
```
执行以上命令在控制台可以获得输出, 对应exchange_info.py的函数说明:
* get_exchange_info_token:  输出可交易的Tokens.
* get_exchange_info_price:  输出Token的交易价格精度, 对应字段为tickSize.
* get_exchange_info_volume: 输出Token的交易数量精度, 对应字段为stepSize.
